Aula 1 - Instalação necessária

Nodejs

Baixar o instalador do Node.js: Acesse o site oficial do Node.js (https://nodejs.org/en/download/) e baixe o instalador adequado para seu sistema operacional (Windows, macOS ou Linux).

https://nodejs.org/en/download/


Executar o instalador: Execute o arquivo baixado e siga as instruções do instalador para instalar o Node.js e o gerenciador de pacotes npm (Node Package Manager).


Verificar a instalação: Abra um terminal ou um prompt de comando e digite node -v e npm -v para verificar a versão do Node.js e do npm instalados.
